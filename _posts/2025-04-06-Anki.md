Study in Anki is limited to the currently selected deck as well as any subdecks it contains.

Answer buttons

- Again (keyboard shortcut 1)
- Hard  (keyboard shortcut 2)
- Good  (keyboard shortcut 3, space, Enter)
- Easy  (keyboard shortcut 4)

When studying, either Space or Enter will show the answer. When the answer is shown, you can use <kbd>Space</kbd> or <kbd>Enter</kbd> to select the Good button. You can use the 1-4 keys to select a specific ease button. 

Many people find it convenient to answer most cards with Space and keep one finger on 1 for when they forget.


## Filtered Decks

To create a filtered deck manually, [↩︎](https://forums.ankiweb.net/t/how-to-review-all-cards-in-the-deck-and-or-other-deck/4400/2)

1. Tools menu > choose Create Filtered Deck.
2. Enter Deck name. Under Search, enter “deck:DECKNAME”, set limit to 9999 cards (to filter out all the cards), enable random selection, deselect “Reschedule cards” and build the deck.
3. If you wish to fetch cards again using the same filter options, <kbd>Rebuild</kbd> the deck.

When a card is moved to a filtered deck, it retains a link to the deck, from which it came. That previous deck is said to be the card’s “home deck”.

It is also possible to move all cards back to their “home decks” at once:

- The <kbd>Empty</kbd> button in the study overview moves all cards in the filtered deck back to their home deck, but does NOT delete the empty filtered deck. This can be useful if you want to fill it again later (using the Rebuild button).

  <img src="{{site.baseurl}}/images/anki filtered deck.png" alt="" style="display: block; margin-right: auto; margin-left: auto; zoom:40%;" />

- **Deleting** a filtered deck does the same thing as “Empty” does, but also removes the emptied deck from the deck list. No cards are deleted when you delete a filtered deck.
