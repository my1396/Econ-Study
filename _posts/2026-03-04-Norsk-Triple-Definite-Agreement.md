本文介绍挪威语中冠词､ 形容词和名词 "三重一致" 的规则。这些词根据性数､ 是否特指进行形变。


## 一重一致

非特指: "adj + substantiv" 的结构中，形容词和名词都要进行性数一致的变化。


| Gender | Singular                   | Plural                      |
| ------ | -------------------------- | --------------------------- |
| Mas.   | en stor bil                | stor**e** biler             |
| Fam.   | ei stor seng               | stor**e** senger            |
| Nut.   | et stor**t** hus           | stor**e** hus               |

--------------------------------------------------------------------------------

## 双重一致

特指: 无形容词。

### That

"den/det/de + definite substantiv" 的结构中，冠词 (指示代词) 和名词都要进行性数一致的变化。

| Gender | Singular                   | Plural                      |
| ------ | -------------------------- | --------------------------- |
| Mas.   | den bilen                  | de bilene                   |
| Fam.   | den sengen                 | de sengene                  |
| Nut.   | det huset                  | de husene                   |

### This

"denne/dette/disse + definite substantiv" 的结构中，冠词和名词都要进行性数一致的变化。

| Gender | Singular                   | Plural                      |
| ------ | -------------------------- | --------------------------- |
| Mas.   | denne bilen                | disse bilene                |
| Fam.   | denne sengen               | disse sengene               |
| Nut.   | dette huset                | disse husene                |

--------------------------------------------------------------------------------

## 三重一致

特指: 有形容词。这是三重一致的完全体。

### That

"den/det/de + definite adj. + definite substantiv" 的结构中，冠词、形容词和名词都要进行性数一致的变化。

| Gender | Singular                   | Plural                      |
| ------ | -------------------------- | --------------------------- |
| Mas.   | den store bilen            | de store bilene             |
| Fam.   | den store sengen           | de store sengene            |
| Nut.   | det store huset            | de store husene             |


### This

"denne/dette/disse + definite adj. + definite substantiv" 的结构中，冠词、形容词和名词都要进行性数一致的变化。

| Gender | Singular                   | Plural                      |
| ------ | -------------------------- | --------------------------- |
| Mas.   | denne store bilen          | disse store bilene          |
| Fam.   | denne store sengen         | disse store sengene         |
| Nut.   | dette store huset          | disse store husene          |
